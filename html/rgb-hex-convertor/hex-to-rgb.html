Hex to RGB converter

It takes input in the form of a hex color code value and converts that value to a RGB value that can be used to specify color in photo editing software.

Photo editing software usually represents color in RGB and therefore if you would like to use the same colors that you use in your html pages in your photo editing software you will need the RGB values for the hex code. This tool allows you to get those values.
